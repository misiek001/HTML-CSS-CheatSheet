This is Cheat Sheet for CSS Selectors and HTML Table elements

Selector Description Example
Single Selector p Select all p element p { font-family:Arial; }
.class Select all elements with class .class { font-family:Arial; }
#id Select element with specific id #id { font-family:Arial; }
Combined Selectors p.class Select all p elements with specific class p.class { font-family:Arial; }
#id p Select all p elements within element with id #id p { font-family:Arial; }
#id p class Select all p elements with class within element with id #id p.class { font-family:Arial; }
p, h2, h3 Select all p, h2, h3 within document p, h2, h3 { font-family:Arial; }

Element Meaning
<table> Main tag for defining table
<thead> Tag for defining Header row
<th> Tag for defining header cell
<tr> Tag for defining table row
<td> Tag for defining table cell
